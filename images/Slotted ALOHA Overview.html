Slotted ALOHA is a network protocol used for sharing a communication channel among multiple users efficiently. It improves upon the original ALOHA protocol by reducing the chance of data collisions.

What Slotted ALOHA Does:

  • Divides time into equal slots (hence "slotted").

  • Users can only send data at the beginning of these time slots.

  • If two users transmit in the same slot, a collision occurs and both packets are lost.

  • If only one user transmits in a slot, the data is successfully received.

Key Features:

  • Better efficiency than Pure ALOHA: Maximum throughput is 37% (vs. 18% for Pure ALOHA).

  • Synchronization required: All users must align their transmissions to time slots.

  • Still probabilistic: No guarantee of success, just improved odds.

Used in:

  • Early wireless networks

  • Satellite communication systems
